## Requirements

+ Ruby 2.3.0
+ RVM (recommended)

## Installation

+ Install SQLite3: `sudo apt-get install sqlite3`
+ Install bundler: `gem install bundler`

## Project Setup

1. Install dependencies and plugins: `bundle`
2. Setup the project: `rake setup`
